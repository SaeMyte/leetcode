# 桶排序 时间复杂度O(n)
之前的所有排序只和两个数怎么比大小有关，均是**基于比较的排序**。
## 不基于比较的排序——计数排序
int []   每一个数字都是员工的年龄
数据范围 0~200
申请一个长度为201的数组，0位置表示0岁员工有多少个，1位置表示1岁员工多少个...
遍历原始数组
都是根据数据状况做的排序，没有基于比较的排序有那么广的应用范围，需要数据状况定义。
